Everything you need

view_kanban

Kanban Board

Organize your work into customizable columns. Any page can become a Kanban board with nested sub-pages as cards.

edit_note

Markdown Notes

Write rich notes with full Markdown support, YAML frontmatter, syntax highlighting via highlight.js, and live preview.

check_circle

Todo Checklists

Create GitHub-style checklists. Toggle checkboxes interactively right from the view mode.

account_tree

Mermaid Diagrams

Render flowcharts, sequence diagrams, and more with Mermaid syntax. Click any diagram to zoom in.

search

Find in Page

Press Cmd/Ctrl+F to search within the current page with overlay highlighting and navigation.

flash_on

Slash Commands

Type / in the editor to quickly insert markdown snippets. Fully customizable via Settings.

image

Image Storage

Paste, drag-and-drop, or upload images. Stored as files in .images/ directory with content hashing.

text_fields

Font Customization

Customize font family and size for the editor, preview, and UI independently via Settings.

folder

Local & Private

Your data lives in plain Markdown files on your machine. Track changes with git.

desktop_windows

Desktop App

Available as a native desktop app via Tauri v2. Works on macOS, Windows, and Linux.

label

Tag System

Categorize pages with tags and filter by them. Full-text search across all pages.

pwa

PWA Support

Install from your browser and use offline. Works with File System Access API.

Get started in seconds

1

Select a Folder

Choose any directory on your machine as your workspace. The app creates a workspace/ subfolder inside.

2

Create Pages

Add notes, kanban boards, and todo lists with Markdown. Use slash commands for quick formatting.

3

Stay Organized

Filter by tags, search content, and customize fonts. Everything is stored as plain markdown files.
